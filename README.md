# hwp-decompress
Bindata scan & decompress

## Requirements
Windows

Python 3 ver

## Install
```bash
python -m pip install -r requirements.txt
```

## Usage
Python file
```bash
python run.py [sample folder path]
```

## Result
save file extension
```bash
.decompress
.hex
```
